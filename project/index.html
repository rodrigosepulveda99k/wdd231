Board Game Shop

Explore, Buy, and Enjoy Board Games

Site Purpose

The site provides a platform for board game enthusiasts to explore and purchase a variety of board games. It also includes reviews, game recommendations, and blogs about board game trends.

Common Scenarios

  • What are the best family board games?
  • Where can I find board game reviews before purchasing?

Color Schema

The primary colors for this site are:

  • Primary Color: #4CAF50 (Green) - Used for buttons, links, and headings
  • Secondary Color: #FFC107 (Amber) - Used for highlights, accents, and call-to-action areas

Typography

The following fonts will be used on the website:

  • Font for Headings: Roboto - Used for all headings and titles
  • Font for Body Text: Open Sans - Used for all paragraph text and descriptions
